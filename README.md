# App de Clima ☁️

Aplicativo de clima desenvolvido em Python utilizando Tkinter e a API da OpenWeather.

O app permite pesquisar qualquer cidade do mundo e visualizar informações climáticas em tempo real de forma simples e visual.

---

# Funcionalidades

* Pesquisa de cidades do mundo inteiro
* Temperatura em graus Celsius
* Umidade do ar
* Pressão atmosférica
* Velocidade do vento
* Horário local da cidade pesquisada
* Tradução de países e continentes para português
* Mudança automática de tema conforme o horário do local
* Interface gráfica feita com Tkinter

---

# Tecnologias utilizadas

* Python
* Tkinter
* OpenWeather API
* Requests
* Pillow
* pytz
* pycountry-convert

---

# Bibliotecas necessárias

Instale as dependências utilizando:

```bash
pip install -r requirements.txt
```

Ou instale manualmente:

```bash
pip install requests
pip install pillow
pip install pytz
pip install pycountry-convert
```
